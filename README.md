### API 간단 요약

---

#### 1. 프로젝트 목록 조회

- **경로**: `GET /api/projects`
- **파라미터**: 없음
- **본문**: 없음
- **응답**: 전체 프로젝트 목록 (배열)

---

#### 2. 사용자 목록 조회

- **경로**: `GET /api/users`
- **파라미터**: 없음
- **본문**: 없음
- **응답**: 전체 사용자 목록 (배열)

---

#### 3. 특정 프로젝트의 작업 목록 조회

- **경로**: `GET /api/tasks`
- **파라미터**: `projectId` (쿼리 스트링)
- **본문**: 없음
- **응답**: 해당 프로젝트의 작업 목록 (배열)

---

#### 4. 새로운 작업 생성

- **경로**: `POST /api/tasks`
- **파라미터**: 없음
- **본문**: 생성할 작업 정보 (JSON)
- **응답**: 생성된 작업 정보 (JSON)

---

#### 5. 작업 상태 수정

- **경로**: `PATCH /api/tasks/:taskId`
- **파라미터**: `taskId` (URL 경로)
- **본문**: 수정할 상태 정보 (JSON)
- **응답**: 수정된 작업 정보 (JSON)
